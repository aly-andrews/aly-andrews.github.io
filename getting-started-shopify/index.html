Essential setup before launch.

Who should read this manual

This manual is for people who have created a Shopify store but have not launched it yet, have limited experience with the Shopify admin and need simple, practical instructions to get a basic store ready to accept orders.

How to use this manual

Start with this overview to understand the five setup areas. Open the Shopify admin in another tab and work through each section in order, following the steps while logged in. Use the References page for troubleshooting, glossary terms and FAQs.

What is Shopify?

Shopify is an ecommerce platform for building and managing an online store. One admin handles products, payments, shipping and checkout. Shopify does not market your business or generate sales. You set up the store, add products and run your own marketing.

Prefer to watch first? Start with this companion video, then use the manual to complete each setup area.

Before you begin

Have a Shopify account and access to the admin. Open this manual alongside the admin and plan 2 to 4 hours for setup across one or more sessions. Have business information ready: address, contact details and basic info. Use a computer or tablet for setup rather than mobile.

If you get stuck, use the References page for troubleshooting and FAQs, search the Shopify Help Center for current screenshots and country rules, or contact Shopify Support for live help.

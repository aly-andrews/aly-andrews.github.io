Getting started on Shopify: Essential setup tasks

A practical manual to help new merchants configure the essentials before launch.

This manual gives new Shopify merchants clear, step-by-step instructions for completing the essential setup work before selling online. It focuses on five areas you must configure for a basic store to accept and fulfill orders.

Who should read this manual

This manual is for people who have created a Shopify store but have not launched it yet, have limited experience with the Shopify admin and need simple, practical instructions to get a basic store ready to accept orders.

How to use this manual

  1. Start with this overview page to understand the five setup areas.
  2. Open your Shopify admin in another tab.
  3. Work through each section page in order, following the procedures while you are logged in.
  4. Use the reference page for troubleshooting, glossary terms and FAQs.

What is Shopify?

Shopify is an ecommerce platform for building and managing your online store. It gives you one admin for products, payments, shipping, storefront settings and checkout.

Shopify does not automatically market your business or generate sales. You still need to set up your store correctly, add your products and run your own marketing plan.

What you will set up

Across the manual you will:

  1. Set up your store settings and basic configuration.
  2. Configure your payment gateway.
  3. Add and organize your products.
  4. Set up your shipping.
  5. Configure your checkout.

Prefer to watch first? Start with this companion video, then use the manual to complete each setup area.

Before you begin

Make sure you have:

  • A Shopify account created and access to your Shopify admin.
  • This manual open alongside your admin.
  • Time set aside to complete setup (2 to 4 hours total, which you can spread over multiple sessions).
  • Your business information ready (address, contact info and basic details).
  • A computer for setup. Tablet is acceptable and mobile is not recommended.

Getting help

If you get stuck while completing the setup steps, you can:

  • Use the troubleshooting guide and FAQs on the reference page.
  • Search the Shopify Help Center for current screenshots and country specific rules.
  • Contact Shopify Support for live help.

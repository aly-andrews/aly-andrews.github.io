# Images Directory

Upload your portfolio images here. The site will automatically use them once you add them to the correct locations.

## Directory Structure

### Portfolio Images (`portfolio/`)
Upload work sample images here:

- `shopify-docs.jpg` - Getting started on Shopify (homepage featured work)
- `video.jpg` - Main video production image (homepage)
- `video-main.jpg` - Large video production image (work page)
- `video-pos.jpg` - POS series
- `video-international.jpg` - International expansion
- `video-migration.jpg` - Content migration

### About Images (`about/`)
- `profile.jpg` - Your professional photo (homepage intro section)

## Image Recommendations

- **Format**: JPG or PNG
- **Size**: 
  - Large images: 1200-1600px wide
  - Medium images: 800-1000px wide
  - Small images: 600-800px wide
- **Optimization**: Compress images before uploading for faster page loads
- **Aspect Ratios**:
  - Portfolio images: 4:3 or 16:10
  - Profile photo: 1:1 (square)

## How to Use

1. Add your images to the appropriate folders above
2. The site will automatically display them (no code changes needed)
3. If an image isn't found, a placeholder will show with instructions

## Tips

- Use high-quality, professional images
- Keep file sizes reasonable (under 500KB when possible)
- Ensure images are well-lit and clear
- Consider using tools like TinyPNG or ImageOptim to compress images
